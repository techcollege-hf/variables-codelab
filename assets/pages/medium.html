Her er opgaver med Animation og JavaScript. Man bruger typisk et event eks. click eller loaded til at triggere en animation.


JavaScript skal skrives i filen: /opgavefiler/js/medium.js

evt. CSS skal skrives i filen: /opgavefiler/css/medium.css

Opgave 1

Du skal her bruge en eventlistner på en button til at starte animation på et andet objekt.
Når du trykker på BØH! skal den grønne kugle animere til højre, væk fra knappen, og ease til stop.

Instruktioner

  1. Opret classes og keyframe sets der kan kontrolere din animation. test evt med hover før du skriver JS.
  2. tilføj en event listner til knappen med id "buhOneButton", der kan toggle din animations klasse ind på elementet med id "ballOne".

resultat her



            JS hint:
                myBuhButton.addEventListener('click',()=>{
                myBall.classList.toggle('animateRight');
                });

            CSS hint:
                .animateRight{
                animation-name: goRight;
                }

Opgave 2

Du skal her bruge en eventlistner på en button til at starte animation på flere objekter.
Når du trykker på BØH! skal de gule kugler animere til højre, væk fra knappen, og ease til stop.

Instruktioner

  1. Opret classes og keyframe sets der kan kontrolere din animation. test evt med hover før du skriver JS.
  2. tilføj en event listner til knappen med id "buhTwoButton", der kan toggle din animations klasse ind på dine elementer med class "balls".

resultat her



            JS hint:
            // i event listner
            for (const myBall  of myBalls) {
                myBall.classList.toggle('animateRight');
            };

            CSS hint:
                .animateRight{
                animation-name: goLeft;
                }

Opgave 3

Du skal her bruge en eventlistner på en button til at starte animation på flere objekter.
Når du trykker på BØH! skal de røde kugler animere skiftevis til højre og til venstre, væk fra knappen, og ease til stop.

Instruktioner

  1. Opret classes og keyframe sets der kan kontrolere din animation. test evt med hover før du skriver JS.
  2. tilføj en event listner til knappen med id "buhThreeButton", der kan toggle din animations klasser ind på dine elementer med class "hardBalls".

resultat her



            JS hint:
            // i event listner brug en global variabel til at holde styr på
            om det er til højre eller venstre du skal toggle på classlist
            

            CSS hint:
                @keyframes goRight{
                from {transform: translateX(0px)}
                to {transform: translateX(500px)}
                }
              
                 @keyframes goLeft{
                from {transform: translateX(0px)}
                to {transform: translateX(-500px)}
                }

Opgave 4
