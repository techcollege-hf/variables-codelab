CSS Animation med keyframes er afhængig af følgende keywords:.


  • @keyframes - Navngivet sæt af instruktioner der estemmer animation.
  • animation-name - Definerer navnet på den @keyframes samling du vil animere igennem.
  • animation-duration - Definerer hvor lang tid din animation tager.
  • animation-delay - Definerer hvor lang tid der skal gå før din animation starter
  • animation-iteration-count - Definerer hvor mange gange din animation skal gentage sig.
  • animation-direction - Definer hvilken vej din animation skal afspilles.
  • animation-timing-function - Samme funktion som på transitions eks. ease
  • animation-fill-mode - hvordan skal det animerede element opføre sig efter animationen er stoppet.
  • animation - shorthand keyword til at definere alle de ovenstående på en linje.

CSS skal skrives i filen: /opgavefiler/css/easy.css

Opgave 1

Instruktioner

  1. Opret en @keyframes samling med navn popUp. der skalerer et element op til 110%
  2. tilføj animation properties til den class der hedder .pop:hover så elementet animerer en gang gennem din @keyframes sæt. på 400ms, med ease.
  3. Din animation skal forblive skaleret op. Sålænge musen er over, og så snappe tilbage når musen flyttes.

Hover her



                    .pop:hover{
                        animation-name: popUp;
                        animation-duration: 500ms;
                        animation-fill-mode:both;
                        animation-timing-function: ease;
                      }
                      
                      @keyframes popUp {
                        from {transform: scale(100%)}
                        to {transform: scale(120%)}
                      }

Opgave 2

Instruktioner

  1. lav en ny @keyframes samling med navn popUpDown. der skalerer op til 120% og dernæst ned til 100% igen.
  2. tilføj animation properties til den class der hedder .popTwo:hover så elementet animerer en gang gennem din @keyframes sæt. på 400ms, med ease.

Hover her



                    @keyframes popUpDown {
                        0% {transform: scale(100%)}
                        50% {transform: scale(120%)}
                        100% {transform: scale(100%)}
