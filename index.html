Animation med keyframes

CSS keyframe animation med transform: translate er en måde at skabe animeret bevægelse af et element på en webside ved hjælp af CSS. Egenskaben "transform: translate" bruges til at flytte et element fra en position til en anden på skærmen, og keyframe animation bruges til at angive start- og slutpunkterne for bevægelsen og timingen af animationen. Ved at definere keyframes ved forskellige procentdele af animationen kan udviklere skabe komplekse animationer, der flytter et element glidende over skærmen eller endda roterer det. Her er et eksempel på, hvordan man kan skabe en simpel animation med transform: translate: 


                /* Definerer elementet, der skal animeres */
                .box {
                width: 100px;
                height: 100px;
                background-color: red;
                position: relative;
                animation-name: flyt;
                animation-duration: 3s;
                animation-iteration-count: infinite;
                }

                /* Definerer animationen */
                @keyframes flyt {
                0% { transform: translateX(0); }
                50% { transform: translateX(200px); }
                100% { transform: translateX(0); }
                }

I dette eksempel vil "box" elementet flytte sig 200px til højre og tilbage til dens oprindelige position over en periode på 3 sekunder. Denne animation gentager sig uendeligt med "animation-iteration-count: infinite".
